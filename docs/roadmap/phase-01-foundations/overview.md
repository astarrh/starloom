# Phase 01 — Foundations

## Development Objectives
- Finalize stable core domain shapes and config schema.
- Lock deterministic seed handling and RNG stream derivation behavior.
- Establish strict/compatible reproducibility modes as implementation constraints.

## Milestone Alignment
- 1. Finalize domain model and config schema.
- 2. Implement seeded RNG stream infrastructure.

## Expected Outcomes
- Core dataclasses and config validators are implementation-ready.
- RNG strategy is fully codified (seed normalization, stream names, version pinning inputs).
- Reproducibility rules are testable and unambiguous for later phases.
